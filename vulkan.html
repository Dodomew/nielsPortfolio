Vulkan (Multithreaded Renderer)

Language: C++11
Engine: custom
Platforms: PC

Created a personal engine dubbed ‘Raven’. It’s goal was to implement a new rendering API and learn more about C++11. Another goal was to focus on its making it as asychronous as possible.

  • Uses the Vulkan API
  • All file reading is done on one thread
  • Update starts as soon as possible instead of waiting for a whole frame to be rendered
  • Implemented ImGUI
All progress can be followed on github: github.com/nielsuiterwijk/vulkan
