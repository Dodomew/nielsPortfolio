ibb & obb

Language: C++
Engine: Phyre Engine
Platforms: PS3, PC (Steam)

ibb & obb is a two player cooperative game set in a puzzle filled world where gravity goes both up and down. You can only succeed by working closely together. Find a friend for some true local co-op couch fun or match up online.

Details

I worked shortly on this project, my main work involved optimizing shaders for the PS3 and implementing a byte serializer for easy packet creation. Besides that I've helped with guiding the game through TRC.
